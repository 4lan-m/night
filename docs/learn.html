About

Night is a high level interpreted language built around safety. It being gradually typed, combined with a robust and safe type system, makes Night a simple yet powerful language.

Safety

A major advantage Night has over other dynamically typed language is its compile time checks.

Whereas a language like Python does not provide type checking and many other declaration checks, Night fills this gap, eliminating hidden runtime bugs and providing safer code.

Type System

Even though Night's dynamically typed, it still features a unique and robost type system that provides strong control to the user.
