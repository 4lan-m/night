Tutorial

Hello World!

For basic output to the console, you can use the print() function. It takes a single parameter of any type, and displays it to the console. Singleline comments are also supported.

Keep in mind that the print() function is buffered, and will only print right before taking in user input or the program's termination.

Variables

To initialize a variable, you can use one of five types: bit (boolean), syb (character), int (integer), dec (float), or str (string).

Variable declaration and assignment are also as expected. However, variable declaration will initialize the variable with a default value depending on its type.

User Input

User input in Night takes the form of the function input(). The function will automatically deduct the type of the user input and return it.

Expressions

Common boolean, arithmetric, and string expressions are supported. They can be used whenever a value is expected. Operator precedence can be found in the reference guide.

Some implicit conversions are also supported. Type int and type dec can be used interchangeably, and any type can concatenate into type str.

Conditionals

Conditionals in Night take use of curly brackets to group code together.

Functions

Functions in Night support all five types, along with a null (void) type. Regular type functions must have a return value, while null type functions do not need a return value.

Note that functions have to be defined before they are used.

Loops

There are two types of loops in Night, the while and for loop. The while loop will only run if its condition evaluates to true, and the for loop will only run a prespecified amount of times.

Unlike the while loop, the for loop's condition is static and cannot change once it's set.

Arrays

To initialize an array, use square brackets after the variable type, and contain your elements using square brackets as well. You can also specify the amount of elements in your array by putting a value inside the square brackets. If an array is not initialized with any or enough elements, then the appropriate default value will be set. Arrays can only contain one type of value.
